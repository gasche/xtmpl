Xtmpl

What is Xtmpl ?

Xtmpl is a small XML templating library for OCaml. Have a look at the reference documentation for details.

Download

Xtmpl is hosted on Github.

Releases

  • 0.2 [2012-08-28]: Minor changes.
  • 0.1 [2012-06-20]: Initial release.

Installation

Xtmpl requires Xmlm >= 1.1.0 to be installed.

To compile and install:

  • tar xvfz xtmpl-XXX.tar.gz
  • cd xtmpl-XXX
  • make all install

This will install the xtmpl package with findlib.
