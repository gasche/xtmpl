Release 0.11 of Xtmpl is available from the download page.

Changes:

  • add ppx_xtmpl: this preprocessor creates functions from template files or strings, with the parameters described in the template. Usage: let doc = [%xtmpl "file.xml"] let doc = [%xtmpl.string {|
    ...
    |} ]
  • some functions renamed (deprecating original ones functions),
  • add Xtmpl.node and Xtmpl.cdata,
  • new Xtmpl_xhtml module, with convenient functions to create Xhtml trees,
  • new Xtmpl_js module in a new package xtmpl.js.
